***

A resonsive web application that gets the users current location and returns covid statistics for the local authority the location is covered by.

***
### How to run

- Docker (required)
- Npm (Optional)

*Using Docker:*
```
docker-compose up --build -d
```

Using Npm:
```
npm run compose
```

The application will be available on port 3000 when the script successfully executes.
